lspplot
=========

A set of plotting stuff and scripts for the plasma code LSP's outputs.

Prerequisites
-------------
* `numpy>=1.10.1`
* `scipy>=0.16.0`
* `lspreader>=0.1.7`
* `pys>=0.0.4`

Install
-------

Using `pip`, and installing locally,

    git pull https://github.com/noobermin/lspplot.git
    cd lspplot
    pip install . --user

Be sure to have installed `lspreader` and `pys` from me.
